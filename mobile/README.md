# mobile

Flutter로 만든 Android 앱. 지금은 "앱이 정상적으로 실행되고 있습니다"라는 문구만 보여주는
최소 화면입니다. Claude가 이 클라우드 세션 안에서 `flutter analyze`, `flutter test`로 이미
검증했습니다.

## 나중에 본인 폰에서 직접 실행해보려면

1. 본인 컴퓨터에 [Android Studio](https://developer.android.com/studio) 설치
2. [Flutter SDK 설치 가이드](https://docs.flutter.dev/get-started/install) 따라 설치
3. 터미널에서:
   ```bash
   cd mobile
   flutter pub get
   flutter run
   ```
   (USB로 연결한 Android 폰 또는 에뮬레이터가 필요합니다)

이 단계는 프로젝트가 더 진행된 후, "실제 폰에서 확인해보기" 단계에서 다시 자세히 안내할게요.
